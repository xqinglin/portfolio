Image Processing

Prediction of surface normals from single image with CNN

Project Summary: Deep learning is an algorithm for feature learning. It can avoid the trouble of manually extracting features, so that the computer can learn the required features on a large scale. The development of multi-layer neural networks is one of the milestones in the history of machine learning. Main job: Review relevant papers and build models based on task requirements. After deciding to use the "hourglass architecture" model, use TensorFlow to build the model in Python. Twenty thousand training set images were trained on Google Cloud, and two thousand images from test set were tested. The resulting average angular error was 0.5.

3D face reconstruction with photometric stereo

Photometric stereo is a fast and accurate 3D surface reconstruction method. The basic principle is to use the mathematical relationship between lighting conditions and the projected imaging to get a three-dimensional model. Main job: Import image sets from the Yale Face database and preprocess the data. Write code to calculate surface normals from images. When recover the surface height from the surface normal vector, a randomness is added to achieve global optimization. Finally, display the model.

3D face reconstruction with photometric stereo
